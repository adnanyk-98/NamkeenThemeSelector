🎨 Custom Colors

How to Use Custom Colors:

  1. Select "Custom" theme from the theme options above
  2. Choose your colors using the color pickers below
  3. Click "Apply Custom Colors" to save and apply
  4. View your posters - colors will be applied automatically!

Color Usage:

  • Primary: Main background gradient and header elements
  • Secondary: Secondary gradient and accent elements
  • Accent: Highlights, borders, and special elements
  • Background: Product pouch backgrounds
